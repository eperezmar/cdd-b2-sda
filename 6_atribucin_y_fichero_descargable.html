6. Atribución y fichero descargable.

Información general sobre este recurso educativo
Título Desarrollo de una aplicación web para la gestión del banco de alimentos
Descripción Esta situación plantea un proyecto realista en el que el alumnado debe desarrollar una aplicación web completa orientada a la gestión y trazabilidad de los productos de un banco de alimentos, integrando funcionalidades como: Gestión de usuarios (administradores, voluntarios, entidades receptoras). Control de entradas y salidas de productos. Notificaciones y alertas sobre caducidad y stock bajo. Consultas e informes sobre movimientos y balances. Seguridad y control de acceso.
Autoría Eugenia Pérez Martínez
